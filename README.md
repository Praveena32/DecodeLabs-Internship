# DecodeLabs-Internship

Simple solutions and exercises for the DecodeLabs internship.

## Description

This repository contains short Java exercises used during the DecodeLabs internship.

## Tasks

### Task 1: Number Guessing Game
- File: `Task1.java`
- Main class: `NumberGuessingGame`
- Description: A number guessing game with up to 10 attempts per round, score tracking, and replay support.
- Run:
  ```bash
  javac Task1.java
  java NumberGuessingGame
  ```

### Task 2: Student Grade Calculator
- File: `Task2.java`
- Main class: `StudentGradeCalculator`
- Description: Prompt for subject marks, calculate average percentage, and assign a grade.
- Run:
  ```bash
  javac Task2.java
  java StudentGradeCalculator
  ```

### Task 3: ATM Simulator
- File: `Task3.java`
- Main class: `Task3`
- Description: A simple ATM interface with balance inquiry, deposit, withdrawal, and exit options.
- Run:
  ```bash
  javac Task3.java
  java Task3
  ```

### Task 4: Currency Converter
- File: `Task4.java`
- Main class: `Task4`
- Description: Convert currencies using fixed exchange rates and a friendly command-line interface.
- Run:
  ```bash
  javac Task4.java
  java Task4
  ```

## Requirements

- Java JDK 11 or newer
